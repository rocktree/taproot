Roadmap Publicly Available

To keep the GitHub issue tracker and to make sure I don't lose site of where this thing is headed, I've put together a roadmap, which is now publicly available within the repo.

At this time, the file will live in the root directory of the project. So, this is the file in the master branch.

Do note, this file will likely change as development continues. So, if you want to see where it lives, check the bloodiest branch. The url to the file will look like this:

https://github.com/seancdavis/sapwood/blob/[version_branch]/ROADMAP.md

This can give you a good idea as to where I'm headed with this project.
